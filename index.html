Como rodar

  1. Coloque esta pasta em qualquer lugar.
  2. Abra um terminal dentro dela e rode: python -m http.server 8000
  3. Acesse no navegador: http://localhost:8000

Abrir o index.html direto (duplo clique) pode bloquear algumas APIs. Use o servidor local recomendado.

Sem servidor e sem banco externo: so HTML+JS com IndexedDB.
